🟣 INS Snap · Installer

Installs the INS snap into MetaMask Flask, then tests .igra resolution.

REQUIRES  MetaMask Flask (separate extension from regular MetaMask — both can coexist)

Installs npm:ins-snap-resolver from the public npm registry.

scanning for wallet extensions via EIP-6963…

After install — test it:

  1. Open MetaMask Flask · switch network to Ethereum Mainnet
  2. Click Send
  3. In the recipient field, type igranetwork.igra
  4. Flask should resolve it to the V2 owner's address (Safe multisig)

Troubleshooting

  • "MetaMask Flask not found" but you have it installed → reload this page after the wallet list populates; if still missing, make sure Flask is enabled in chrome://extensions/
  • Install dialog shows but resolution does nothing → check Flask is on Ethereum Mainnet (snap is gated to eip155:1 and eip155:38833)
